# 🛒 Catat Belanja (Shopping Tracker)

Aplikasi berbasis web sederhana namun fungsional untuk mencatat pengeluaran belanja harian. Didesain dengan antarmuka modern dan responsif untuk memudahkan penggunaan langsung dari smartphone saat sedang berbelanja.

## ✨ Fitur Utama
- **Digitalisasi Catatan:** Gantikan bon kertas yang mudah hilang dengan catatan digital yang rapi.
- **Kategori Berwarna:** Identifikasi jenis belanjaan (Sembako, Jajanan, Dapur, Mandi) secara visual dengan label warna-warni.
- **Statistik Otomatis:** Pantau total pengeluaran dalam 7 hari terakhir dan 30 hari terakhir dengan animasi angka yang interaktif.
- **Input Fleksibel:** Mendukung perhitungan barang **Satuan** (pcs/bungkus) maupun **Kiloan** (kg/ons/gram).
- **Aksi Cepat (Long-Press):** Tekan lama pada item untuk melakukan **Edit** atau **Checklist** (mencatat ulang barang yang sama dengan cepat).
- **Fitur Pencarian:** Cari riwayat belanjaan lama hanya dengan mengetik nama barang.

## 🛠️ Teknologi
- **HTML5 & CSS3** (Custom Glassmorphism UI)
- **Tailwind CSS** (Utility-first styling)
- **Vanilla JavaScript** (Lightweight logic)
- **LocalStorage API** (Data tersimpan di memori browser pengguna)

## 🚀 Cara Penggunaan
1. **Tambah Barang:** Klik tombol `+` di pojok kanan bawah.
2. **Pilih Kategori:** Tentukan jenis barang agar laporan keuangan lebih rapi.
3. **Pilih Tipe:** Gunakan mode "Satuan" atau "Kiloan" sesuai kebutuhan.
4. **Edit/Ulang:** Tekan dan tahan (long-press) pada kartu belanjaan untuk memunculkan menu aksi.
5. **Reset:** Gunakan tombol "Reset" di bagian atas jika ingin menghapus seluruh data.
